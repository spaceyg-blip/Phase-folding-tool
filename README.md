# Phase Folding Tool

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/drive/1SmS2N8MbbNyLOg8-Yxar6lYLv5r9OidV?authuser=1#scrollTo=s7moZ2onsXX5)

Astronomy project for phase folding and visualization of stellar light curves using Python.



## Features

- Load TESS light curves
- Apply phase folding
- Visualize phased stellar signals
- Save phase diagrams
- Explore stellar variability



## Dataset

Mission: **TESS (Transiting Exoplanet Survey Satellite)**

Target used:

**AB Dor**

Approximate period:

**0.514 days**
